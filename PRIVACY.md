# Privacy Policy — The-Jagoda-Toolkit

This plugin runs entirely locally inside Claude Code on the user's machine.

**Data collection:** None. The plugin does not collect, transmit, or store any
user data, prompts, code, or telemetry.

**Network activity:** None initiated by this plugin. The optional `rtk-rewrite.sh`
hook invokes `rtk` if installed; `rtk` is a third-party tool with its own
privacy policy
(<https://github.com/rtk-ai/rtk>).

**Third-party services:** None. Companion tools (CodeGraph, rtk, caveman) are
installed and run separately by the user under their own terms.

**Contact:** Open an issue at <https://github.com/Jagoda11/the-jagoda-toolkit/issues>
